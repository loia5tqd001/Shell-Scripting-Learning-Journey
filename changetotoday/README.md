# Change git history commits to today


#### Usage

```
./changetotoday [number of commits]
./changetotoday 5  # make the last 5 commits's histories today
```


#### Environment
- OS: Linux
- MacOS
- Bash environment for Windows


#### Technology

- `Bash shell`
